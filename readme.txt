== Description ==

**Charts And Graphs Manager** You can create different charts in this plugin.You can create a chart and use its shortcode on your custom page.

This plugin allows you to add custom labels and custom data.

**Charts And Graphs Manager** Background colors and border colors are available in this plugin.

*Easy to use
*Add Multiple values
*Background Color Chage Available
*Add Multiple Chart
*Add/Remove Data

= How to Use Custom Chart =
Select Chart Types. Add The Data And Add That Shortcode To Your Custom Page.
